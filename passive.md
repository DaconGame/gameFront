# 패시브 설계

## 방향

지금 강화는 유닛 추가 / 공격력 / 공속 / 이속 / 회복 / 최대HP라 **단순 수치**다.
패시브는 수치보다 **플레이 방식이 바뀌는 것**이 맛있다. 1순위는 **직업별 패시브 카드** — 업그레이드 선택지에 섞이고, 전사면 전사 느낌이 강해지는 식.

---

## 설계 원칙 2가지 (중요)

### 1. 조합(시너지)과 역할을 분리한다

`report.md`의 롤토체스식 시너지와 효과가 겹치면 안 된다.

- **조합(시너지)** = 같은 직업 N명 모으면 **자동**으로 붙는 정체성 효과 (관통/장판/보호막/넉백 등 "직업의 기본 색깔").
- **패시브 카드** = 드래프트에서 **직접 고르는** 증폭기. 시너지를 베끼지 말고 **조건·트리거가 있는 능동적인 것** 위주.

따라서 아래 효과는 **시너지로 흡수**하고 패시브 풀에서는 뺀다:
회전 베기(반경↑), 관통 사격, 잔류 화염, 보호막, 용병 결속.

### 2. "숨은 수치형"보다 "플레이가 바뀌는 형"

- ❌ 숨은 숫자 (재미 약함): 마력 증폭(+15px), 방패 숙련(-15%), 약점 표식(+15%)
- ✅ 플레이가 바뀜 (맛있음): 전장의 함성, 전투 리듬, 처치 본능, 응급 본능, 집중 사격, 피의 칼날, 정화

---

## 패시브 풀 (직업별)

### 검사

- **피의 칼날**: 근접 처치 시 HP 2 회복 (공격적 근접 보상) ✅
- **전장의 함성**: 근처 적 3마리 이상이면 공격력 +20% (몰아서 싸우는 의사결정) ✅
- 방패 숙련: 피격 데미지 15% 감소 (수치형)
- 회전 베기: 근접 스플래시 반경 +20px → **검사 시너지로 이동 권장**

### 궁수

- **집중 사격**: 같은 적 연속 공격 시 데미지 증가 ✅
- 약점 표식: 맞춘 적 3초간 받는 피해 +15% (수치형)
- 관통 사격: 첫 적 뒤 1명 추가 타격 → **궁수 시너지로 이동 권장**

### 마법사

- **연쇄 폭발**: 마법으로 적 처치 시 작은 추가 폭발 ✅
- 마력 증폭: 광역 반경 +15px (수치형)
- 잔류 화염: 폭발 지점에 장판 데미지 → **마법사 시너지로 이동 권장**

### 성직자

- **축복**: 회복받은 뒤 3초간 이동속도 +10% ✅
- **정화**: 일정 주기마다 주변 적 둔화 ✅
- 보호막: 회복 시 HP 만렙이면 임시 보호막 → **성직자 시너지로 이동 권장**

### 공용 (직업 고정 완화)

- **처치 본능**: 적 처치 시 모든 쿨타임 소폭 감소 ✅
- **응급 본능**: HP 30% 이하일 때 방어력 증가 ✅
- **전투 리듬**: 공격하지 않고 2초 지나면 다음 공격 강화 ✅
- 용병 결속: 같은 직업 2명 이상이면 공격력 증가 → **시너지 시스템으로 흡수**

## 1차 구현 추천 (6개)

"구현 가벼움 + 플레이 변화 체감"을 같이 잡은 셋.

| 패시브 | 이유 |
|---|---|
| **피의 칼날** (킬→HP2) | 근접 공격성 보상, 생존과 직결 |
| **전장의 함성** (적 3+ → 공격력 +20%) | "몰아서 싸운다" 의사결정 |
| **처치 본능** (킬→전체 쿨감) | 스노우볼, 전 직업 호환(공용) |
| **응급 본능** (저체력 방어↑) | 위기 역전, 난이도 상향과 시너지 |
| **전투 리듬** (2초 후 강타) | 카이팅/치고빠지기 유도 |
| **연쇄 폭발** (마법 킬→추가 폭발) | 광역 학살감, 마법사 색 강화 |

> 참고: `회전 베기`는 기존 스플래시 45→65px로 **구현이 가장 싸서** 빠른 첫 삽 용으로는 합리적. 다만 위 원칙대로면 **검사 시너지(2명)** 로 넣는 게 깔끔하다.

## 패시브 등장 시점 (조건부 + 단계 개방)

처음부터 풀에 다 섞으면 초반 군대 뼈대(고용)를 못 만들고 카드 풀도 희석된다. 단계적으로 연다.

| 구간 | 카드 구성 | 패시브 |
|---|---|---|
| **웨이브 1~4** (도입) | 고용 + 기본 강화만 | ❌ 없음 — 군대 뼈대부터 |
| **웨이브 5 보스 클리어** | 보스 보상 = 첫 패시브 픽 | ✅ 보유 직업 패시브 3택1 |
| **웨이브 6~19** (일반) | 고용·강화·패시브 혼합 | ✅ 보유 직업 + 시너지 2명↑ 조건 필터 |
| **웨이브 10·15 보스 클리어** | 보스 보상 = 강한 패시브/조합 강화 | ✅ 마일스톤 특별 픽 |

### 등장 조건 (결합)

```
(웨이브 ≥ 5) AND (해당 직업이 파티에 있음) AND (가능하면 시너지 임계 2명↑)
```

- 안 뽑은 직업의 패시브는 안 뜸 → 희석 방지.
- 시너지 2명 모으면 그 직업 패시브가 풀에 등장 → **"조합 → 패시브 강화"** 로 연결.

### 왜 보스 클리어 시점인가

- 보스(5웨이브)면 이미 4번 픽 → 직업 색이 잡혀 있어 관련 패시브를 줄 수 있음.
- 보스는 난이도 스파이크라 보상으로 강해지는 흐름이 자연스러움 (라운드 보상 느낌).
- "보스 잡았더니 새 능력 해금" → 해방감/성장 비트가 또렷.

### 빈도·중첩

- 매판 패시브 X. **2~3웨이브에 1번 꼴** 또는 보스 보상 중심.
- **한 번 고른 패시브는 1회 해금(중복 제거)** → 후반 폭주 방지.

---

## 꼭 정해둘 디테일

1. **적용 범위**: 패시브는 플레이어 본체만이 아니라 **해당 직업 전원**에 적용 (조합과 맞물려 빌드가 됨). 현재 `MERC_COMBAT` 공유 구조라 직업 단위 적용이 자연스러움.
2. **중첩 규칙**: 같은 패시브 재등장 시 1회 해금(중복 제거).
3. **드래프트 필터**: 패시브는 보유 직업이 파티에 있을 때만 등장 (롤토체스도 보유 시너지 위주로 뜸).
